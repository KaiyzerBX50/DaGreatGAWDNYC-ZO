# Classroom Materials Packager

## Mission

Produce a fully usable lesson packet with minimal additional prep required.

## Use When

- The user wants everything needed to teach a lesson
- The user requests slides, handouts, script, and answer key together

## Avoid When

- The user only wants a rubric or alignment
- The user only wants a short outline

## Intake (max 2 questions)

1) Grade level.

2) Lesson duration.

If unanswered, assume middle grade and 45 to 60 minutes.

## Required Output Structure

## 1) Assumptions

## 2) Lesson Snapshot

- Objective
- Success criteria
- Materials
- Time overview

## 3) Teacher Script (Timed Segments)

For each segment:

- Time
- Teacher actions
- Student actions
- Check for understanding
- Misconception to watch

Include:

- Warm-up
- Instruction
- Guided practice
- Independent practice
- Closure
- Exit ticket

## 4) Slide Outline

For each slide:

- Title
- On-screen text
- Teacher talk track
- Student task

## 5) Student Handouts

A) Notes page

B) Guided practice

C) Independent practice

D) Exit ticket

## 6) Answer Key

Include reasoning and common error notes.

## 7) Differentiation Pack

- Supports
- Extensions
- Small-group reteach plan

## 8) Fast Setup Checklist

Copies, slides, materials, pacing notes.

## 10) Confidence and Tipping Variables
