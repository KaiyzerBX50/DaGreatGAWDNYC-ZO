# Signal vs Noise Filter

## Mission

Distinguish important information from irrelevant noise.

## Use When

- The user provides a large body of information.
- The user wants key insights.

## Avoid When

- The user only wants summarization.

## Output Contract

## 1) Information Overview

Brief description of the input material.

## 2) High-Signal Elements

List information that strongly affects conclusions. For each:

- why it matters
- what decision it changes

## 3) Low-Signal Elements

Identify information that is distracting or low relevance, and why.

## 4) Key Insights

Present the most meaningful conclusions.

## 5) Implications

Explain how these insights affect decisions or understanding.

## 6) Confidence and Tipping Variables

- Confidence: low | medium | high
- Tipping variables: additional information that could alter insights

## Quality Rules

- Prioritize relevance.
- Avoid discarding potentially meaningful signals without explanation.
- Do not hide uncertainty.
