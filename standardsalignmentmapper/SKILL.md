# Purpose

Unpack standards into teachable skills, map lessons and assessments to them, and identify coverage gaps and misalignments.

# Input

Ask:

- Standard or learning outcome (paste text, state the standard code, or name it)
- Grade band or course
- Lesson or unit name
- What is the assessment, project

If unanswered, proceed with the user's wording and make assumptions explicit.

# Method

## 1) Standard Unpacking

Produce:

- Verbs and skills
- Concepts and content
- Cognitive demand level
- Required prior knowledge

## 2) Learning Targets

Convert unpacked standard into 3-5 measurable, observable targets.

## 3) Proficiency Scale

Build a 4-level scale:

- Below standard
- Meets standard
- Exceeds standard
- Advanced

Write descriptors tied to student work.

## 4) Alignment Table

Map lesson/unit tasks to targets:

- Task or lesson element
- Target alignment
- Depth match: below, match, above
- Gap notes

## 5) Coverage Map

Summarize:

- Targets covered well
- Targets weakly covered
- Targets missing

# Output Format

## Standard Unpacking

## Learning Targets

## Proficiency Scale

## Alignment Table

## Coverage Summary

## Recommended Fixes

# Quality Bar

- Targets are measurable and observable
- Proficiency descriptors cite student evidence
- Alignment identifies gaps and specific fixes
