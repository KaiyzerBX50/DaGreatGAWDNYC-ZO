# Consequence Engine

## Purpose

Map first, second, and third order effects, feedback loops, and irreversibility risks.

## Use When

- The user wants long horizon thinking
- The system has multiple stakeholders or feedback loops
- The action could change incentives, behavior, or structure

## Avoid When

- The user only wants a numeric forecast
- The question is purely factual and near term

## Clarifying Questions

Ask up to 2 if needed:

1) What time horizon matters most: months, years, decades?

2) Who are the main stakeholders affected?

If unanswered, proceed with stated assumptions.

## Method

### 1) Define the Initiating Action

- One sentence action statement
- Scope and boundary: what is included, what is excluded

### 2) First Order Effects

- Direct effects
- Immediate behavior changes

### 3) Second Order Effects

- Effects caused by first order outcomes
- Incentive changes, substitution, spillover

### 4) Third Order Effects

- Structural change: institutions, market structure, norms
- Path dependence risks

### 5) Feedback Loops

Identify loops as:

- Reinforcing loop
- Balancing loop

For each loop:

- Trigger
- Amplifier or dampener
- What breaks the loop

### 6) Nonlinear and Tail Risks

- Tipping points
- Cascades
- Worst case but plausible events

### 7) Reversibility and Mitigations

- Reversibility rating: easy, moderate, hard
- 2 to 4 mitigation actions with owners and timing

## Output Format

### Action and Scope

### First Order Effects

### Second Order Effects

### Third Order Effects

### Feedback Loops

### Tail Risks

### Reversibility and Mitigations

### Confidence and Tipping Variables

## Quality Bar

- Concrete mechanisms, not generic lists
- Separate plausible from speculative
