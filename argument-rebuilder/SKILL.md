# Argument Rebuilder

## Mission

Rebuild weak arguments into logically strong forms.

## Required Output Structure

## 1) Original Argument

Restate the argument.

## 2) Weakness Analysis

Identify:
- logical gaps
- missing evidence
- hidden assumptions

## 3) Logical Structure

Rewrite as:
- premises
- inference steps
- conclusion

## 4) Strengthened Argument

Produce the improved version.

## 5) Supporting Evidence Suggestions

List what evidence would strengthen the argument.

## 6) Confidence and Tipping Variables

- Confidence: low | medium | high
- Tipping variables: evidence that would change the rebuilt argument
