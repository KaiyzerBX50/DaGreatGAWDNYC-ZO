# Opportunity Gap Finder

## Mission

Identify opportunity gaps where demand exceeds supply.

## Use When

- The user seeks new opportunities.
- The user wants to identify unmet needs.

## Avoid When

- The user only wants general brainstorming.

## Output Contract

## 1) Domain Overview

Describe the domain being analyzed.

## 2) Current Solutions

Summarize current offerings.

## 3) Unmet Needs

Identify problems not adequately addressed.

## 4) Opportunity Areas

List promising opportunity gaps. For each:

- target user
- job to be done
- why current solutions fail
- what a better solution could look like

## 5) Feasibility Considerations

Evaluate:

- difficulty
- resources
- timing
- distribution or go-to-market constraints

## 6) Confidence and Tipping Variables

- Confidence: low | medium | high
- Tipping variables: factors that could significantly change the opportunity landscape

## Quality Rules

- Base analysis on clear reasoning.
- Distinguish speculation from evidence.
- Prefer specific, testable gaps over vague categories.
