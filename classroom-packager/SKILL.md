# Classroom Materials Packager

## Purpose

Generate a cohesive set of classroom materials in one response: teacher script, student handouts, practice, checks for understanding, and a slide-by-slide outline.

## Use When

- The user wants everything needed to teach a lesson with minimal extra prep
- The user has a topic, standard, text, or lesson idea and wants materials built around it
- The user wants print-ready student pages plus a teacher-facing plan

## Avoid When

- The user only needs one artifact (only a quiz, only a rubric, only an email)
- The user has no topic and is asking for general ideas

## Inputs

If the user provides them, use them. If not, proceed with assumptions and label them:

- Grade or learner profile
- Time available (lesson length)
- Topic or standard
- Text, problem set, or source material (optional)
- Class format (whole group, stations, workshop, lab)

## Output Contract

Always produce the packet in the order below and keep it classroom-ready.

## Packet Assembly Protocol

### 1) Assumptions

List any assumptions made about grade, time, prior knowledge, and constraints.

### 2) Lesson Snapshot

- Objective in student-friendly language
- Success criteria (3 to 5 bullets)
- Materials list
- Timing outline

### 3) Teacher Script

Write a short, teachable script with:

- Opening and framing (what students will do and why)
- Modeling (worked example or think-aloud)
- Guided practice prompts
- Questions to ask students
- Anticipated misconceptions and quick fixes
- Closure and exit ticket directions

Format as a minute-by-minute or segment-by-segment script.

### 4) Slide Outline

Provide a slide-by-slide plan with:

- Slide title
- On-screen text
- Teacher talk track
- Student task
- Check for understanding moments

Keep slides minimal and readable.

### 5) Student Handouts

Create print-ready handouts with headings and whitespace:

A) Notes page

- Key vocabulary
- Mini examples
- Space for student notes

B) Guided practice

- 3 to 6 items with space to work
- Hints optional, placed after items

C) Independent practice

- 6 to 12 items
- Mix difficulty: basic, on-level, challenge

D) Exit ticket

- 2 to 4 questions tied to success criteria

### 6) Answer Key

Provide a complete answer key for guided practice, independent practice, and exit ticket.

Include short rationales where misconceptions are common.

### 7) Differentiation Pack

Provide:

- Supports: sentence frames, word banks, worked example, chunking plan
- Extensions: 2 to 4 challenge tasks
- Small group plan: who to pull, what to reteach, a 10 minute micro-lesson outline

### 8) Fast Setup Checklist

Provide a checklist the teacher can follow right before class:

- Copies to print
- Board or slide prep
- Materials staging
- Quick room setup
- What to watch for during work time

## Quality Bar

- Materials align tightly to objective and success criteria
- Student pages are copy-paste printable
- Every practice set has an answer key
- Script and slides match the handouts with no contradictions
