# Arbitrage Detector

## Purpose

Find high return skill investments by combining user fit, market demand, scarcity, and durability.

## Use When

- The user wants a learning plan linked to economic outcomes
- The user is considering a pivot or specialization
- The user wants ranked options with a roadmap

## Avoid When

- The user is asking for a list of trending buzzwords
- The user refuses to discuss constraints, time, or interests

## Clarifying Questions

Ask up to 2 if needed:

1) What is your current skill stack and experience level?

2) What constraint is tightest: time, money, or risk tolerance?

If unanswered, proceed with stated assumptions.

## Method

### 1) Current Stack Snapshot

- Technical skills
- Domain knowledge
- Proof of work
- Preferences and constraints

### 2) Market Lens

Rate each candidate skill area on:

- Demand strength
- Talent saturation
- Automation exposure
- Cross industry portability
- Time to competence

Use simple labels: low, medium, high.

### 3) Opportunity Generation

Produce 6 to 10 candidate directions:

- Adjacent expansions
- Complementary pairings
- Role based packages

### 4) Rank Top 3

Rank using a clear rubric:

- Time to first payoff
- Scarcity advantage
- Compensation upside
- Durability
- User fit

### 5) 8 Week Plan for Top Pick

Provide:

- Weekly focus
- Project output each week
- How to show proof publicly or to employers
- One checkpoint at week 4 with pass fail criteria

### 6) Confidence and Tipping Variables

Provide:

- Confidence: low, medium, high, with one sentence why
- Tipping variables (3 to 7): the specific facts that would change the top 3 ranking
- For each tipping variable: what you would look for to update it

## Output Format

### Current Stack

### Candidate Opportunities

### Ranked Top 3 with Rationale

### 8 Week Plan for Top Pick

### Proof of Work Outputs

### Confidence and Tipping Variables

## Quality Bar

- Concrete outputs, not course lists only
- Ranking logic visible and repeatable
