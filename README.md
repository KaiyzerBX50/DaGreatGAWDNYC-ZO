# Zo Teacher OS

Prompt builder and sample exports for creating differentiated, standards-aligned K–12 lessons.

**Live demo:** https://dagawdnyc.zo.space/teacher-os

## Features

- **Prompt builder** – Customize by grade, subject, topic, and duration
- **Sample exports** – DOCX, PDF, PPTX, and CSV formats ready for classroom use
- **Differentiation built in** – Supports multiple learner profiles, assessments, and classroom routines
- **Standards-safe** – Include official standard codes (ELA, Science, Math)
- **One-click generation** – Copy prompt, paste into Zo Chat or your preferred AI app

## How It Works

1. **Describe the class** – Grade, subject, topic, standards, and constraints
2. **Copy the prompt** – Use the generated prompt in your AI chat (Zo, ChatGPT, Claude, etc.)
3. **Download exports** – Get DOCX lesson plan, one-page PDF, PPTX slides, and CSV trackers

## Included Exports

- Lesson plan (DOCX)
- One-page plan (PDF)
- Slides outline (PPTX)
- Student materials, handouts, and parent letters
- Assessment checklist, rubric, and standards tracker
